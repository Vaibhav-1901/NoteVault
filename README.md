# NoteVault

A clean, minimal, dark-themed notes app focused on distraction-free writing with instant auto-save and real-time filtering.

→ [note-flow-eight-neon.vercel.app](https://note-vault-eight-neon.vercel.app)

## 🚀 Features
- Sidebar + editor layout — note list on the left, writing area on the right
- Tag notes (work, personal, ideas, etc.) and filter by tag from the sidebar
- Search bar that filters as you type
- Auto-save, so you never have to think about it
- On mobile, the sidebar slides in as a drawer from the toggle button in the top left

## Tech Stack

- Frontend: React,Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB


⚠️ Note

First load may be slow due to Render free tier (backend sleeps when idle)
